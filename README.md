# Messenger Anon v0.4

Минималистичный анонимный текстовый мессенджер.

## Принципы

- только логин и пароль;
- только текст;
- файлы/картинки/видео — ссылками;
- без онлайн-статуса;
- без read receipts;
- без контактов из телефона;
- без рекомендаций;
- без геолокации;
- минимум логов, без текста сообщений;
- сервер должен знать минимум.

## Что есть в v0.4

- регистрация;
- вход;
- личные чаты;
- WebSocket live-сообщения;
- история сообщений;
- счётчики непрочитанных;
- автообновление списка чатов;
- блокировка пользователя;
- удаление аккаунта с удалением личных переписок;
- Docker/Docker Compose;
- сервер сам отдаёт веб-клиент.

## Запуск на Windows 11

```powershell
cargo run
```

Открыть:

```text
http://127.0.0.1:8080
```

## Запуск через Docker

```bash
docker compose up --build
```

## Проверка двух пользователей

1. Открой обычное окно браузера.
2. Зарегистрируй `First`.
3. Открой приватное окно.
4. Зарегистрируй `Second`.
5. Найди пользователей по логину и напиши сообщение.

## Важно

Это всё ещё dev/MVP. Сейчас `user_id` передаётся с клиента, поэтому для настоящего интернета нужен следующий шаг: нормальные сессии/токены и HTTPS.

## v0.4 — групповые беседы

Добавлено:

- единый список личных и групповых чатов;
- создание группы по названию и логинам участников через запятую;
- сообщения в группах через WebSocket;
- история групповых чатов;
- счётчики непрочитанных для групп;
- без онлайн-статусов и без read receipts;
- текст сообщений не пишется в серверные логи.

### Как проверить группы

1. Запусти сервер: `cargo run`
2. Открой 3 окна: обычное, приватное, другой браузер.
3. Зарегистрируй `First`, `Second`, `Third`.
4. От `First` создай группу, например:
   - название: `test`
   - участники: `Second, Third`
5. Напиши в группу. Сообщение должно прийти всем подключённым участникам.
